Help & quick guide

Create a diagram

Edit Mermaid code or drag a shape into the preview. On touch devices, drag its icon or grip. Changes render and save automatically.

Edit visually

Click a node to edit its shape, label, colors, border, or convert it to an image. Drag nodes between groups or together to create nesting.

Add connected nodes

Hover over a node and click its circular arrow. Subgraph arrows create nodes outside the container.

Create an edge

Click the source node, choose Create edge, then click the destination node. Use the banner’s Cancel button or Esc to stop. Click the new arrow to edit its label, line, color, or endpoints.

Navigate

Drag empty preview space to pan. Scroll the mouse wheel or pinch with two fingers to zoom. Fit, Center, theme, and Fullscreen controls are at the bottom.

Diagrams and recovery

Use Diagrams to manage projects and open History to restore local snapshots. MMD imports or downloads source, while Export creates SVG, PNG, JPG, or PDF files.

Keyboard shortcuts

Ctrl/Cmd+Enter render · F fit · Ctrl/Cmd+Shift+E export · Ctrl/Cmd+Z undo · Ctrl/Cmd+Shift+Z redo · Delete delete selected · Esc cancel or close.

About Mermaid Flow Editor

Version 1.0

Diagramming that stays fast, private, and local

Mermaid Flow Editor is a standalone visual workspace for creating Mermaid flowcharts. Write Mermaid code directly or build visually with shapes, nodes, connectors, and subgraphs while the preview updates live.

Version 1.0 highlights

Visual flowchart editing

Add, connect, restyle, group, and move nodes without leaving the preview.

Offline and private

No backend, account, analytics, CDN, or required network access.

Local diagram library

Create and manage multiple diagrams with automatic saving and version recovery.

Flexible export

Download Mermaid source or export polished SVG, PNG, JPG, and PDF files.

Productive workspace

Resizable panels, templates, themes, zoom, pan, fit, fullscreen, and search.

Accessible interaction

Keyboard navigation, focus-managed dialogs, screen-reader labels, and touch controls.

Your work stays yours. Diagrams and recovery snapshots remain in this browser unless you explicitly download or export them.

A quick note for touch devices

Touch controls are fully supported, but for the smoothest experience—especially when arranging detailed diagrams—we recommend using a desktop or laptop with a mouse.
